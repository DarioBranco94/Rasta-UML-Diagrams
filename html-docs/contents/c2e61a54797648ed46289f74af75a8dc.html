Diagram

Description

This sequence diagram describes a possible interaction of the user with the components involved in showing the POI information. The sequence can be described as: 1. the user request to the frontend the webpage showing the list of POIs 2. the frontend makes a request to the POI-TOI manager that returns the list of POIs in the database. 3. The POIs list is displayed on the frontend. 4. The user can navigate through the results and select a POI of interest. 5. A view of the POI is shown. 6. The user can read info and optionally can click on links redirecting to external resources to add details on the reported information.
