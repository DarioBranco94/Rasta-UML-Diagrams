Edit/Delete POI

UMLUseCase
RASTA :: Requirement Analysis :: Generic Use Cases :: Edit/Delete POI

Description

This functionality of the system allows to modify a POI in the system. The admin creates the POI, filling the required information as geographical location, category, description etc and later can edit the POI to consider new data. This function can be extended with a semantic enrichment.
