Description

Description: Multiplatform Mobile and Web applications o see, place, and share 3D objects. The application use gps also to retrieve location-based 3D objects. The manager, the first one can access shared 3D Objects, the latter one can upload 3D objects, place the model in the surrounding pace and save the position information in 3D-DB if there is a GPS coverage or relative positioning information in NoSQL Realtime DB through Google Cloud Anchors API, if there isn’t GPS coverage

Dependencies: Depending on 3D-DB and NoSQL Realtime storage

Interfaces: CRUD for 3D-DB, WebSocket for real-time NoSQL and Google Cloud Anchors API
