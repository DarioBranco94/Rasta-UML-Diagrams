Semantic Enrichment

UMLUseCase
RASTA :: Requirement Analysis :: Generic Use Cases :: Semantic Enrichment

Description

This functionality of the system allows to enrich the POI in the system. with semantic links. While the admin is creating or editing the POI, filling the required information as geographical location, category, description etc, the system will analyze the fields to identify semantic links to recognized entities. This function requires the text to be annotated with semantic links linking to external resources.
