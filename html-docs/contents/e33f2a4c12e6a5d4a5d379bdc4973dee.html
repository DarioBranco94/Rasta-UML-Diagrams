Annotation and Metadata

UMLComponent «Database»
RASTA :: Logical Architecture :: DatabaseAndRepository :: «Database»Annotation and Metadata

Description

Description: The component stores the metadata and (semantic) annotations of digital objects using standard and interoperable formats.

Dependencies: The component is used by semantic annotators to store the generated metadata, by the digital archive exporting engine to export/import metadata in different formats, by the annotated content viewer to visualize the annotations together with the annotated content.

Interfaces: It offers REST APIs.
