Description

Description: This component is a web-based GUI for tourists that want to get personalized itineraries from the TRS. It exposes facilities for entering topic preferences, time constraints, and transportation mean constraints. It also allows users to visualize and filter POIs on a geographical map, and facilities for consultation and exploration of the personalized itineraries returned by the system.

Dependencies: It directly uses the following components: POI-TOI Manager, POI and TOI – User Profiler Matchmaker, Travel Planner. It is directly used by Tourist (a tourist user).

Interfaces: It exposes a graphical user interface with the main facilities described above.
