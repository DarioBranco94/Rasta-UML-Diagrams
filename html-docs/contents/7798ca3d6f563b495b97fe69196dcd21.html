Description

This sequence shows how the decision maker can obtain reports about the tourism flow analysis. The request starts from the PMS backoffice frontend where the decision maker has access to the POIs of interest. Selecting a POI from the administrative frontend the user activates a request also on the flow analyzer that will query the tourism flow DB to retrieve data about the selected POI. The result will generate dashboards where the decision maker can query and navigate data and results interactively.
