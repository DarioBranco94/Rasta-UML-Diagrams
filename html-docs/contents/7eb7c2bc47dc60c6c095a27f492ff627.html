Read POI Information

UMLUseCase
RASTA :: Requirement Analysis :: Generic Use Cases :: Read POI Information

Description

The user can read information about a selected POI. This use case can be activated by selecting a POI of the list on the frontend. While reading the description and information about the POI, the user can follow the links in the text permitting to navigate to an external resource (e.g. wikipedia, external website). This use case is optional since the user can be satisfied about the provided information on the frontend.
