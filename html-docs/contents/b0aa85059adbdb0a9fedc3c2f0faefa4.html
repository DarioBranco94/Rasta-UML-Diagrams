Compute POI -TOI scores

UMLUseCase
RASTA :: Requirement Analysis :: Generic Use Cases :: Compute POI -TOI scores

Description

The function of asking the system to automatically (re)compute the scores associated with each pair <POI,TOI>. This operation may take some time, depending on the amount of data stored in the system; hence it should be performed asynchronously with respect to the online interaction.
