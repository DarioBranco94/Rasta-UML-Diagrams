Select POI

UMLUseCase
RASTA :: Requirement Analysis :: Generic Use Cases :: Select POI

Description

This use case represents the selection of a single POI by the user. The user can select a POI of interest after filtering and refining the list returned by the system. After the selection the user can read POI information navigating to the POI webpage.
