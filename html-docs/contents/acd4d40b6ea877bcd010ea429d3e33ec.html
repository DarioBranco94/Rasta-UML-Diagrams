NoSql RealTime

UMLComponent «Database»
RASTA :: Logical Architecture :: DatabaseAndRepository :: «Database»NoSql RealTime

Description

Description: This non-relational Database stores information about the users and the relative position in the surrounding space of the 3D objects that they have placed within the augmented reality App. It provides for real-time access which will allow one user to share a positioned 3D object in a space captured by the camera and a second user to immediately view its location.

Dependencies: No dependencies

Interfaces: Realtime API to retrieve shared positional information of 3D objects
