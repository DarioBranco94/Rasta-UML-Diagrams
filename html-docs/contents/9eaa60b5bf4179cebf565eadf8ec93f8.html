Internal Domain Ontologies

UMLComponent «Ontology»
RASTA :: Logical Architecture :: DatabaseAndRepository :: «Ontology»Internal Domain Ontologies

Description

Description: Internal repository of domain ontologies containing the definition of scene objects, locations and actors. These ontologies contain, therefore, the elements that the story composer must use to compose stories by means of special graphic palettes

Dependencies: This component is used by the 'Story and Content Instancies Semantic Matchmaker'.

Interfaces: It offers REST APIs.
