Description

This sequence shows how the tourism flow analysis is enabled. A tourist visits a POI that is infrastructured with IoT sensors detecting people approaching it. The IoT sensor automatically and continuously sends raw data about the presence to the Flow DB. The system admin configures the Tourism Flow component to analyze periodically or continuously data. This acts as a cron job that will analyze data and aggregate it.
