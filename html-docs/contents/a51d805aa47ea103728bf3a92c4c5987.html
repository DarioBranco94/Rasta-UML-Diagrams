Story and Content Visualizer

UMLComponent «Component»
RASTA :: Logical Architecture :: LogicalComponents :: «Component»Story and Content Visualizer

Description

Description: This component is responsible for the delivery of stories and content to end users (tourists). The component allows the user to choose how the content is to be fruited (by scene, by action or by character). For example, if the chosen viewing mode is 'by character', the tool returns a navigable graph consisting of all the scenes in which that specific actor appears

Dependencies: It uses the Semantic Engine directly, and is used directly by the end user (tourist).

Interfaces: It exposes a very intuitive graphical interface, aimed at end-user use
