Description

This sequence demonstrates the semantic enhancement applied to the POI management. The user after selected a specific POI can activate the semantic enhancement function that will trigger the analysis of the text looking for semantic links. If the engine finds semantic links then returns the text annotated with the semantic links. The user can select the ones he/She accepts and store the updated text into the DB.
