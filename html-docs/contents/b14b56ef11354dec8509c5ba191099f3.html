Filter POIs

UMLUseCase
RASTA :: Requirement Analysis :: Generic Use Cases :: Filter POIs

Description

This use case permits to refine the list of POIs displayed on the frontend. If the list includes many results the user may want to refine the results by considering his/her preferences. A categorization of the POI can offer a effective way of filtering them. The selection of a POI is the same of the unfiltered list.
