Description

This component represents the main user interface for managing information about POIs and TOIs. It can expose both graphical and textual commands for either visualizing properties of POIs/TOIs or for updating POI/TOI information or for creating new POIs/TOIs in the POI-TOI DB. Particularly important is the possibility of using the POI-TOI frontend to manually insert POIs information that cannot always be automatically extracted from public sources (e.g., the ideal time needed for visiting a POI); or again for enriching the description of TOIs based on the specific geographical context in which they must be used. The component also offers functionalities to run automatic computations of POI-TOI scores on subsets of POIs and TOIs.
